# Kona Yote
A platform where accessibility and local delivery are central. It suggest that you can find anything from every corner of Kenya, whether it's fresh products or house utilities-all just a few click away

## Features
- Cart system
- Use of REST APIs
- Javascript cookie authentication
- Authentication(Sign-up)

## In-development
- Checkout system
- Full authentication(Sign-up & Sign-in)
